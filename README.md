# C-plus-plus
Этот репозиторий содержит различные программы на C++, написанные для выполнения разных задач. Ниже приведено описание каждой программы.
Описание программ
perevod cm to inch and back Eyukin.cpp
Описание: Программа для конвертации сантиметров в дюймы и обратно.
int Sum Eyukin.cpp
Описание: Программа для вычисления суммы всех целых чисел в заданном диапазоне.
Home Eyukin.cpp
Описание: Программа для отображения домика с использованием символов.
chemodan Eyukin.cpp
Описание: Программа для подсчета всех возможных комбинаций цифр для трехзначного кода.
TemplateSelectSortFunctionsEyukin.cpp
Описание: Программа для сортировки выбора с использованием шаблонов функций.
TemplateOverLoadFunctions Eyukin.cpp
Описание: Программа с перегруженными шаблонными функциями для поиска максимума.
TemplateLineSearch.cpp
Описание: Программа для линейного поиска в массиве с использованием шаблонов.
TemplateInsertSortFunctionsEyukin.cpp
Описание: Программа для сортировки вставками с использованием барьерного элемента (стража).
Temperature Eyukin.cpp
Описание: Программа для анализа температур за месяц.
Switch Lesenka Eyukin.cpp
Описание: Программа для расчета скидки на основе суммы покупки.
SwapEyukin.cpp
Описание: Программа для обмена значениями трех переменных.
ShakerSort Eyukin.cpp
Описание: Программа для Шейкерской сортировки массива.
Prime numbers Eyukin.cpp
Описание: Программа для вывода простых чисел от 2 до 100.
OverLoadFunctions Eyukin.cpp
Описание: Программа с перегруженными функциями для поиска максимума.
ModuleEyukin.cpp
Описание: Программа для вычисления модуля числа с использованием макросов.
Masssive2D grade Eyukin.cpp
Описание: Программа для подсчета среднего балла студентов.
Masssive2D grade Eyukin 1.0.cpp
Описание: Улучшенная версия программы для подсчета среднего балла студентов с классификацией.
Functions Prime Eyukin.cpp
Описание: Программа для определения, является ли число простым или составным.
Function_And_Massive_2_Eyukin.cpp
Описание: Программа для замены минимального и максимального элементов в двумерном массиве.
Function_And_Massive_1_Eyukin.cpp
Описание: Программа для изменения порядка элементов в массиве.
Eyukin descent ladder.cpp
Описание: Программа для расчета скидки на основе суммы покупки.
Dice 4.0 Eyukin.cpp
Описание: Программа для игры в кости.
CountGreater Eyukin.cpp
Описание: Программа для подсчета количества чисел в массиве, которые больше заданного.
Circle Eyukin.cpp
Описание: Программа для вычисления площади круга и длины окружности.
Chemodan Eyukin 1.0.cpp
Описание: Программа для подсчета количества целых чисел с двумя одинаковыми цифрами.
Chemodan Eyukin 0.0.cpp
Описание: Программа для подсчета количества целых чисел с двумя или тремя одинаковыми цифрами.
Calculator2.0.cpp
Описание: Программа-калькулятор для выполнения арифметических операций.
Calculator1.0.cpp
Описание: Простая версия калькулятора.
BinarySearchR Eyukin.cpp
Описание: Программа для бинарного поиска в массиве с использованием рекурсии.
